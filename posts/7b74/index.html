Markdown教程

发表于2023-10-03|更新于2023-10-03|博客
|字数总计:1.1k|阅读时长:3分钟|阅读量:|评论数:

前言

由于最近搭建了hexo框架的博客，而hexo使用的是Markdown格式的文章，所以在这记录下Markdown的基本格式

Markdown基本格式

标题

使用 # 标记

段落

两端文字间隔一个空行

字体

分割线

在一行中使用3个及以上的 * - _ 来建立分割线

删除线

在文字的两端加上海两个波浪线 ~~

下划线

下划线可以通过HTML的<u>实现

脚注

列表

Markdown 支持有序列表和无序列表。

无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格，然后再填写内容：

有序列表使用数字加 . 表示

列表嵌套

列表嵌套只需在子列表中的选项前面添加四个空格即可

区块

区块引用是在段落开头使用 > 符号

区块也是可以嵌套的，内层的区块增加 > 即可

区块使用列表

列表中使用区块

在列表内放进区块，那么就需要在 > 前添加四个空格的缩进

代码

如果是段落上的一个函数或片段的代码可以用反引号把它包起来（` ）

如果代码中包含`

代码区块

代码区块使用三个` 包裹,并可以在前面的三个` 后指名语言

如果代码区块中包含三个` ， 那么你需要用6个`来标记代码快的前后

链接

高级链接

我们可以通过变量来设置一个链接，文档末尾给变量赋值

图片

当然，你也可以像网址那样对图片网址使用变量

设置图片大小

Markdown目前只能使用普通标签设置图片大小

表格

Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。

对齐方式

  • -: 设置内容和标题栏居右对齐。
  • :- 设置内容和标题栏居左对齐。
  • :-:设置内容和标题栏居中对齐。

转义

Markdown 使用了很多特殊符号来表示特定的意义，如果需要显示特定的符号则需要使用转义字符，Markdown 使用反斜杠转义特殊字符

Markdown 支持以下这些符号前面加上反斜杠来帮助插入普通的符号
